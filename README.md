# (앱 이름)

## 앱 정의(ADS)
이 앱은 ‘갬성캠핑에 특화된’ 캠퍼들에게 사진 공유를 통한 캠핑 정보와 커뮤니티를 제공하는 앱입니다.

왜냐하면 인증샷을 자주 찍는 감성 컨셉의 캠퍼들에게 본인의 캠핑 사진을 직접 자랑할 수 있는 사진 공유 채널을 제공하고, 뷰가 우수한 캠핑장 정보를 공유할 수 있도록 하여 감성캠핑 라이프에 편리함을 더해주기 때문입니다.
